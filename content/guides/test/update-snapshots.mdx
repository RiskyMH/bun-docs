---
title: Update snapshots
fullTitle: Update snapshots in `bun test`
mode: center
---

Bun's test runner supports Jest-style snapshot testing with `.toMatchSnapshot()`.

```ts title="snap.test.ts"
import { test, expect } from "bun:test";

test("snapshot", () => {
  expect({ foo: "bar" }).toMatchSnapshot();
});
```

---

The first time this test runs, Bun writes a snapshot file to a `__snapshots__` directory alongside the test file.

```txt title="File Tree"
test
├── __snapshots__
│   └── snap.test.ts.snap
└── snap.test.ts
```

---

To regenerate snapshots, use the `--update-snapshots` flag.

```sh title="terminal"
$ bun test --update-snapshots
bun test v$BUN_LATEST_VERSION (9c68abdb)

test/snap.test.ts:
✓ snapshot [0.86ms]

 1 pass
 0 fail
 snapshots: +1 added # the snapshot was regenerated
 1 expect() calls
Ran 1 tests across 1 files. [102.00ms]
```
