---
title: Get entrypoint path
fullTitle: Get the absolute path to the current entrypoint
mode: center
---

The `Bun.main` property contains the absolute path to the current entrypoint.

```ts tab="foo.ts"
console.log(Bun.main);
```

```ts tab="index.ts"
import "./foo.ts";
```

---

The printed path is the file executed with `bun run`.

```sh title="terminal"
$ bun run index.ts
/path/to/index.ts

$ bun run foo.ts
/path/to/foo.ts
```
