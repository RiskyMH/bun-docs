---
title: import.meta.dir
fullTitle: Get the directory of the current file
mode: center
---

Bun provides a handful of module-specific utilities on the [`import.meta`](/docs/runtime/module-resolution#import-meta) object.

```ts title="/a/b/c.ts"
import.meta.dir; // => "/a/b"
```
